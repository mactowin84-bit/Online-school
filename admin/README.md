# Панель администратора для онлайн-репетитора

Минималистичный интерфейс управления для онлайн-репетиторства (5-9 классы).

## Основные разделы

### 1. Дашборд (Обзор)
- Карточки с ключевыми метриками:
  - Всего учеников
  - Занятий сегодня
  - Ожидаемый доход (KZT)
  - Заявки на пробный урок
- Графики:
  - Доход по месяцам (линейный график)
  - Нагрузка по дням недели (столбчатая диаграмма)

### 2. Управление расписанием (Schedule CRM)
- Интеграция с Google Calendar
- Таблица со списком занятий
- Функция "Проверить окна" - автоматическая проверка свободного времени
- Статусы занятий (Подтверждено/Ожидание)

### 3. База учеников
- Список учеников с полями:
  - Имя
  - Класс (5-9)
  - Контакт (WhatsApp/Telegram)
  - Статус оплаты (Оплачено/Долг)
- Возможность добавления/редактирования/удаления

### 4. Финансовый блок
- Сводка: доход за месяц, ожидаемые оплаты
- Список транзакций Kaspi Pay
- Кнопка "Сформировать ссылку на 7 000 ₸" с автокопированием

### 5. Редактор контента
- Редактирование описаний предметов
- Управление ценами
- Загрузка методических материалов

## Технологии

- HTML5, CSS3, JavaScript
- Chart.js для визуализации данных
- Google Calendar API для интеграции расписания
- Адаптивный дизайн

## Настройка

### Google Calendar
В файле `index.html` замените `YOUR_CALENDAR_ID` на ваш ID календаря.

### Kaspi Pay
В файле `admin-script.js` в функции `generateKaspiLink()` замените `YOUR_MERCHANT_ID` на ваш ID мерчанта Kaspi.

## Стиль

- Цветовая схема: серый (#2c3e50), белый (#fff), синий (#3498db)
- Минималистичный дизайн с боковым меню
- Адаптивность для планшетов и мобильных устройств

## Backend интеграция

Для полноценной работы требуется backend для:
- Хранения данных учеников
- Управления расписанием
- Обработки платежей
- Загрузки файлов

Рекомендуемые технологии: Node.js + Express, Python + Flask/Django, или Firebase.

## Запуск

Откройте `admin/index.html` в браузере для локального просмотра.
